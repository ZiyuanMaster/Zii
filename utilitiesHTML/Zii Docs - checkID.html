Perkenalan

Sebelumnya ini merupakan project solo dari Author Ziyuan dan saya selaku Author hanya mengandalkan website yang dapat hosting basic HTML secara gratis. Oleh karena itu, lingkungan menjadi tidak memadai. Ini mengakibatkan Author tidak dapat merancang kode Node JS yang bisa secara langsung memeriksa Username atau In-Game Name (IGN) kalian.

Tapi jangan khawatir, Author sudah menyediakan cara lain agar kalian dapat melakukan hal yang serupa hanya dengan satu aplikasi saja!

HTTP Shortcuts

Ya, nama aplikasinya adalah HTTP Shortcuts! Disini kalian dapat melakukan permintaan HTTP yang dilengkapi fitur Variables dan Script untuk penggunaan yang lebih bervariasi.

Sebelumnya, Author akan menjelaskan sedikit tentang HTTP Request (Permintaan HTTP), HTTP Request adalah suatu mekanisme pengiriman data pada protokol HTTP yang digunakan pada saat melakukan pengiriman data dari sisi client ke sisi server. Singkatnya, kalian dapat menerima sepenggal database yang disediakan server tersebut hanya dengan permintaan saja! Namun perlu diingat, permintaan yang kalian berikan harus serupa dengan yang disediakan server tujuan, jika permintaan melenceng dari apa yang disediakan, maka server tersebut hanya akan mengembalikan nilai eror!

Tidak perlu membuang waktu lebih lama lagi, Author sudah menyediakan link untuk mendownload aplikasi HTTP Shortcuts.

Impor Dan Eksekusi

Bisa dibilang cukup mudah untuk mengimpor koleksi games di aplikasi HTTP Shortcuts dan Author sudah menyiapkan koleksi Check ID Games untuk semua games yang ada di Zii Store loh! Tapi Author tegaskan kepada kalian untuk memastikan sudah menginstall aplikasi HTTP Shortcuts. Selanjutnya kalian hanya perlu memilih game yang kalian inginkan pada tabel di bawah ini, tekan tombol "OK" dan koleksi tersebut akan secara otomatis ditambahkan ke dalam koleksi kalian.

Setelah koleksi Author ditambahkan ke dalam koleksi kalian, yang perlu dilakukan hanya kembali ke menu utama HTTP Shortcuts, klik game yang diinginkan dan kemudian mengisi User ID (dan Server jika ada). Dengan begitu hasilnya akan muncul dalam beberapa saat!

Kesimpulan

Berikut kesimpulan dari langkah-langkah menggunakan aplikasi HTTP Shortcuts:

Download aplikasi HTTP Shortcuts,

Pilih game yang diinginkan dari tabel,

Tekan tombol "OK" dan kembali ke menu utama,

Pilih game dari koleksi HTTP Shortcuts,

Mengisi User ID (dan Server jika ada),

Hasil akan muncul dalam beberapa saat.
